Creazione di un nuovo campo - Opzioni generali

Nella creazione di un nuovo campo il primo step è quello di definirne le opzioni generali.

Clicca sul pulsante nel menu contestuale per accedere alla maschera della sua creazione.
  • Nome:Inserisci il nome del campo;
  • ID: Il sistema assegnerà automaticamente un ID che potrai modificare successivamente;
  • Uso: ovvero l'utilizzo che intendi fare del campo. Questo valore ti servirà per l'eventuale definizione di gruppi.
Se attivi questa opzione, il campo sarà strutturato per contenere una formula, scritta in linguaggio Java.
