I campi contatti

È possibile creare un numero illimitato di campi per far coincidere la struttura del proprio database con quella dei file oggetto dell'importazione.

Sulla piattaforma un campo può essere condiviso, reso cioè disponibile in tutti i database che verranno creati o esistere solo all'interno del database in cui è stato creato.

Per accedere alla gestione dei campi contatti segui il percorso:

Contatti>nome_del_database>gestione database>campi contatti.

Sarà visualizzato l'elenco dei campi contatti propri tuo database.
Cliccando sull'etichetta: Mostra tutti i campi condivisi a questo elenco si aggiungeranno anche icampi condivisi da tutti i database.
