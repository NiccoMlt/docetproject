Selezione del file

Il primo step della procedura di importazione è la scelta del file da importare.

  1. Seleziona il file da importare o indica da quale URL potrà essere scaricato;
  2. Scegli il "Tipo di operazione" da eseguire. Se è la tua prima importazione ed il tuo database è vuoto, questa opzione non sarà disponibile, altrimenti potrai scegliere tra:

    • Importare nuovi contatti e aggiornare i contatti esistenti;

      • Se un contatto non è presente nel database sarà aggiunto;
      • Se un contatto è già presente nel database i suoi dati verranno aggiornati.

    • Importare nuovi contatti;
      Saranno aggiunti solo i contatti non presenti sul database.
    • Aggiornare contatti esistenti;
      Si procederà all'aggiornamento dei soli contatti già esistenti sia sul file che sul database.
    • Sospendere contatti iscritti;
      I contatti che sul database hanno il campo Stato del contatto valorizzato come iscritto vedranno il valore di questo campo trasformato in sospeso.
    • Disiscrivere i contatti iscritti
      Tutti i contatti presenti sia nel file da importare e nel database saranno disicritti dal database.
    • Riattivare i contatti sospesi. Tutti i contatti presenti nel file e sospesi nel database saranno riattivati.

  3. Accetta la clausola in base alla quale dichiari che hai il consenso per importare i contatti contenuti in questo file.
  4. Seleziona il pulsante in alto a destra nella pagina ed accedi allo step successivo.
