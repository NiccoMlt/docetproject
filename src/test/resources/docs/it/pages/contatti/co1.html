La gestione del database

Tutti i contatti vengono raccolti all'interno di un database. La piattaforma consente di crearne un numero illimitato.

Deve esistere sempre almeno un database. Al primo accesso ne è già disponibile uno di default chiamato "Mydatabase".

Per accedere alla gestione del database seleziona la voce Contatti nel menu principale e verrai indirizzato all'elenco di tutti i database disponibili.
Come noterai nell'elenco è presente un database chiamato Masterdatabase, cioè un "superdatabase" in cui confluiranno automaticamente tutti i contatti che saranno aggiunti in qualsiasi altro database.

Per creare un nuovo database, dalla dashboard seleziona il pulsante Nuovo database in alto a destra e procedi al suo Setup di base.

  • Inserisci il Nome del database;
  • Inserisci una descrizione facoltativa;
  • Seleziona i flussi di attività che desideri attivare sul database corrente (In questa fase ti consigliamo di creare tutti i flussi proposti.)
  • Seleziona le lingue in cui saranno mostrati i flussi.

Una volta definiti i parametri di base clicca sul pulsante "Crea" in alto a destra.
