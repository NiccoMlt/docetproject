Campi contatti - Il tipo di dati

Il tipo di campo determina il tipologia di dati che questo potrà contenere.

I tipi di dati disponibili sono:

  • Campo testo
  • Campo elenco valori
  • Campo si/no
  • Campo numero
  • Campo data

Campo testo

Il campo conterrà valori alfanumerici strutturati secondo una delle seguenti modalità:

  • Testo;
  • Testo multilinea;
  • Indirizzo email;
  • Password;
  • Numero di telefono (Generico, cellulare o fax);
  • Cap;
  • Codice fiscale;
  • Partita iva;
Per ognuna delle opzioni del campo di testo potrai definirne la lunghezza minima e massima nel campo Lunghezza.

Campo elenco valori

Con il campo elenco valori puoi creare una lista di opzioni tra le quali il contatto potrà effettuare la sua scelta.
Selezionando l'opzione Permetti selezione multipla i contatti potranno scegliere più di un valore contemporaneamene.

La lista dei valori disponibili per il campo sarà visualizzata nella finestra Valori disponibili.
Per aggiungere nuovi valori, seleziona la voce Aggiungi un valore e nella finestra che compare inserisci il primo dei valori che desideri compongano la tua lista e clicca su Aggiungi. Ripeti poi l'operazione per tutte le opzioni che desideri aggiungere.
Una volta terminato clicca su Chiudi. Nel caso di:
  • Province;
  • Sigle di province;
  • Regioni;
  • Nazioni;
esistono già delle liste disponibili che puoi utilizzare selezionando il pulsante Aggiungi valori da una lista e scegliendo quella che desideri caricare come elenco di opzioni disponibili.

Campo si/no

Consente di scegliere esclusivamente tra un valore "si" e "no".

Campo numero

Dovrai specificare se si tratta di un numero intero o decimale.

Campo data

Dovrai specificare se il campo conterrà solo una data o una data più un orario.
