Definizione parametri file excel

Il secondo step della procedura di importazione è la definizione dei parametri del file excel che si sta importando.

  • Scegli se la prima riga del file excel che stai importando contiene l'intestazione delle colonne.
    Se non attivi questa opzione, il contenuto della prima riga del file excel sarà trattato come tutti gli altri campi del file.
  • Se all'interno del file che si sta importando sono presenti campi di testo che hanno uno o più spazi all'inizio o alla fine, attivando questo controllo saranno eliminati.
Clicca su: in alto a destra per procedere allo step successivo del processo di importazione
