Mappa i campi del file in campi del database

Il terzo step della procedura di importazione è la mappatura dei campi del file con quelli del tuo database. In altre parole, per evitare la perdita di informazioni devi fare in modo che la strutura del file che stai importando coincida con quella del tuo database.

I database della piattaforma hanno già numerosi campi preconfigurati, ma &egravè molto frequente il caso in cui se ne debbano creare di nuovi per poter accogliere tutte le informazioni conservate nel file di origine dei dati.

Nel terzo step devi predisporre tutti i campi che utilizzerai nel tuo database.
Utilizzando il pulsante la piattaforma effettuerà un'associazione automatica tra i campi del file e quelli del database.

I campi associati con successo saranno contrasseganati dall'icona di una matita. Quelli per i quali è richiesta un'associazione manuale saranno contraddistinti dal simbolo di "Modifica campo".

Puoi creare rapidamente un'associazione scegliendo uno dei campi proposti nel menu a tendina a sinistra del simbolo di modifica campo.
Se il campo di cui hai bisogno non fosse invece disponibile dovrai crearlo cliccando sull'icona di Modifica campo che ti farà accedere alla pagina in cui inserire tutte le opzioni necessarie.
Nota: Se vuoi maggiori informazioni sulla creazione e configurazione dei Campi contatti consulta questo link.
Una volta che ad ogni campo presente nel tuo file corrisponderà un campo specifico nel tuo database, clicca su in alto a destra nella pagina per procedere allo step successivo.
