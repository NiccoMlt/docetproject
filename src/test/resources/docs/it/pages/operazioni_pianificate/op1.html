Creazione di un'Operazione pianificata

La creazione di tutte le Operazioni pianificate prevede un setup generale e, se necessario, una successiva configurazione dei parametri specifici.

Per accedere alla loro gestione, entra nella pagina del database per il quale intendi crearle. Dal menu laterale, seleziona la voce "Operazioni pianificate" nella sezione Gestione database:
Accederai in questo modo alla pagina con l'elenco delle Operazioni pianificate esistenti.
