Esportazione automatica dei contatti

Questa operazione esporta in automatico, in un nuovo file, tutti i contatti presenti in un database o in un gruppo.

Procedi attraverso gli step elencati nel menu laterale:

  1. Proprietà
  2. Sorgente dati
  3. Tracciato file
  4. Formato del file
  5. Upload dei file

Proprietà

È la sezione delle Opzioni generali descritta in precedenza.

Sorgente dei dati

È possibile esportare solo i contatti che rispondano al criterio di un filtro impostato. Una volta definito il filtro, utilizza il pulsante "Conta contatti" per conoscerne il numero esatto.

Tracciato file

Configura la struttura del file che sarà esportato.
Nell'esempio riportato sarà creato un file con due colonne, la prima chiamata "Email", la seconda "Cellulare".

Formato del file

Definisci tutti i parametri del file che sarà esportato.
Nel Nome del file è possibile impiegare anche dei placeholders per rendere automatica l'assegnazione del nome al proprio file.
Con le opzioni: Puoi:

  • Rimuovere gli spazi superflui all'inizio ed alla fine del file che stai creando.
    Se un campo, per esempio, al posto di cominciare con un carattere alfanumerico inizia con uno o più spazi, questi saranno automaticamente eliminati. Lo stesso tipo di operazione sarà poi applicata anche ad eventuali spazi presenti dopo l’ultimo carattere.
  • Comprimere il file in formato .zip.
  • Inserire le intestazioni delle colonne nella prima riga.

Upload del file

L'ultimo passo per la configurazione di questa operazione è la definizione del percorso in cui sarà esportato il file generato.
Le soluzioni possibili sono tre:
  • FTP di questo account: l'operazione caricherà il file sull'FTP locale dell'account che sta effettuando l'esportazione.
    La configurazione dell'FTP di piattaforma può essere raggiunta seguendo il percorso del menu Gestione>Sistema>FTP.
    In questa sezione è possibile creare nuove cartelle e caricare file. Una volta configurato il tuo servizio FTP seleziona, mediante il tasto "browse", il percorso in cui sarà depositato il file esportato.
  • File server: Il file sarà archiviato sul file server di un FTP non residente sulla piattaforma.
    Per configurare questo servizio segui il percorso Gestione>Sistema>Server FTP remoti. Nella pagina che compare seleziona il pulsante "Nuovo server" e configura tutti i parametri del tuo FTP remoto, quindi nella sezione di configurazione della tua operazione definisci la cartella in cui sarà esportato il file creato.
  • Dropbox: Il file sarà archiviato su una cartella di Dropbox.
    Dopo aver installato e configurato l'app che consente di utilizzare i servizi di Dropbox, (la guida alla configurazione è disponibile all'interno della pagina dell'app sulla piattaforma), inserisci nel campo "Cartella dropbox" il percorso in cui esportare il file una volta creato.
  • Scegli l'opzione "Sovrascrivi il file se esiste già", se desideri sovrascrivere un file già presente e con lo stesso nome, nella cartella di destinazione.

Esecuzione dell'operazione

Una volta impostati tutti i parametri, seleziona il tasto "Salva" in alto a destra nella pagina.
Se desideri lanciare l'operazione immediatamente, clicca su "Esegui ora" in alto a destra. Ti verrà chiesta conferma dell'avvio immediato dell'operazione.
Al suo termine un report dettagliato ti informerà sul suo esito.
Nota: Ti consigliamo di tenere l'operazione in “Staging” finchè non avrai completato il suo setup ed effettuato tutti i test necessari.
