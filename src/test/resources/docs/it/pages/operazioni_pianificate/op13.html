Eliminazione dati tabella personalizzata

L'eliminazione di dati da una tabella personalizzata è un'operazione molto delicata che andrebbe effettuata solo da utenti esperti.

All'interno della sezione "Operazione" va anzitutto selezionata la tabella sulla quale effettuare la query di selezione dei dati. La procedura esegue periodicamente la query per ricercare i record eliminabili e poi procede con l'eliminazione stessa. La query deve ritornare i valori della primary key della tabella dei record eliminabili.
Ad esempio: 
SELECT idproduct from PRODUCTS WHERE creationdate <= today(-15).
