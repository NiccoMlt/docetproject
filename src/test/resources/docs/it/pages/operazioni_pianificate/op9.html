Sospendi contatti che non ricevono

Questa operazione automatizza la sospensione di tutti quei contatti che non ricevono comunicazioni.
Non prevede l’importazione di un file ma agisce direttamente sul database o su un gruppo.

Oltre al setup di base, l'unica sezione da configurare è quella dell' "Operazione" in cui va definito il Metodo di calcolo: ovvero Se conteggiare:
  • il numero totale di bounce per ogni contatto;
  • quanti bounce consecutivi occorrono perché il contatto sia sospeso.
    • È possibile indicare una soglia minima di bounce perchè un contatto possa essere sospeso ed indicare quale valore assumerà il campo "Causa di sospensione" sul database una volta eseguito l'aggiornamento.
