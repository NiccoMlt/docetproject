Importazione di un file

La procedura più comune per popolare il proprio database è quella di importare un file contenente la lista dei contatti da aggiungere.

È possibile importare i seguenti tipi di file:

  • File di Excel;
  • File di testo in formato .csv o .txt;
  • File in formato .zip.

  1. Seleziona il file da importare
  2. Scegli il "Tipo di operazione". Se è la tua prima importazione ed il tuo database è vuoto, questa opzione non sarà disponibilie
  3. Accetta la clausola in base alla quale dichiari che hai il consenso per importare i contatti contenuti in questo file.
  4. Seleziona il pulsante "Avanti" in alto a destra nella pagina e definisci i parametri del file che stai importando, poi clicca ancora una volta su "Avanti" .
