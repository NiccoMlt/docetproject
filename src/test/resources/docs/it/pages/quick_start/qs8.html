Verifica dei dati e fine processo

L'ultimo passaggio, prima dell'importazione definitiva dei contatti nel datatabase, è la verifica dei dati.
Se l'esito della verifica è positivo potrai procedere all'importazione definitiva dei contati.

La piattaforma esegue una verifica del file prima che questo sia effettivamente importato e genera un report preventivo all'importazione dello stesso.

Con il pulsante "Scarica CSV" si ottiene un file di testo contenente tutti gli errori di importazione riscontrati.
Se sei soddisfatto dei risultati ottentuti clicca sul pulsante "Avvia" in altro a destra per concludere il processo di importazione.
