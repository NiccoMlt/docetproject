Associazione automatica dei campi

Dopo aver caricato il file dei contatti, devi fare in modo che i campi presenti sul tuo file coincidano con quelli presenti sul tuo database.

Se i campi coincidono, ti basterà selezionare il pulsante "Associa in automatico" ed il software genererà sul database tutti i campi necessari, altrimenti dovrai procedere ad associarli manualmente mediante l'icona di "Modifica del campo" mediante la quale potrai creare i campi di cui hai bisogno.
