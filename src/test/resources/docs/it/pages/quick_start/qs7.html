Campi non mappati

Alcuni campi del tuo database contengono valori non direttamente importati da un file.

In questa sezione puoi impostare i valori di default per:

  • Valore del campo Iscrizione;
  • Valore del campo Provenienza contatti;
  • Valore del campo Riceve email/SMS/Fax;
Puoi eventualmente aggiungere ulteriori campi da valorizzare in automatico mediante il pulsante "Aggiungi campo". In questa fase puoi anche scegliere se iscrivere automaticamente i contatti in un gruppo statico o dinamico.
