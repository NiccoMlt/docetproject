Login

Il primo passo per accedere alla piattaforma è quello di inserire le tue credenziali di accesso.

  1. Collegati all'indirizzo della pagina di accesso;
  2. Clicca il pulsante "Login" nella parte superiore della pagina;
  3. Nella maschera di login alla piattaforma inserisci le tue credenziali;
  4. Clicca sul pulsante "Log In".

Nota: Ricorda di creare password complesse, contenenti cifre, numeri e caratteri speciali e di cambiarle con una certa frequenza!
